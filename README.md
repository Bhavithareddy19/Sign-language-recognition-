# Sign Language Recognition

A real-time Sign Language Recognition system using Python, OpenCV, and Machine Learning.

## Features

- Real-time hand gesture detection
- Sign prediction
- Deep learning model
- Live camera recognition

## Technologies Used

- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## Run the Project

```bash
pip install -r requirements.txt
python inference.py
```

## Project Goal

To recognize sign language gestures in real time and improve communication accessibility.
